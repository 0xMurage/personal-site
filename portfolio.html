Events Management System

0

A system (dubbed ploti bambika) that allows events managers to add/edit events into the system where attendees can book tickets and pay all through the system. The system currently supports M-Pesa payments with more integrations to be added. Attendees can search, share, and book tickets for various events. Buying tickets in bulk for a single event is also supported up to an allowed limit. A printable ticket containing a unique QR code is generated on successful payment. The QR code is used to easily and quickly validate a ticket.
